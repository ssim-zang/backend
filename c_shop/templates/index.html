C : Computer
애 : 사랑 애
랑 : 모임

씨애랑이란 C : COMPUTER / 애 : 사랑 애 / 랑 : 모임 이라는 뜻으로 1993년에 개설된 한림대학교 소프트웨어학부 학술동아리입니다. 최근 30주년을 맞이한 현재 씨애랑 동아리는 약 130명의 동아리원들과 함께 합니다.

임베디드, 앱, 웹, AI, VR/AR 게임, 보안 분야의 팀으로 나뉘어 공부하고 있습니다. 동아리 운영진과 팀장을 맡은 동아리원의 주도하에 기초 언어 교육, 특강, 프로젝트 개발 활동을 진행합니다. 소프트웨어 전시회에 작품을 제출하는 것이 최종 목표 입니다.
